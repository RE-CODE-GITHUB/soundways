API

Clarity provides a simple API that you can use to interact with the plugin.

Events

You can bind event handlers using the built in on and once functions.

    
        var clarity = $('#myPlayer').ttwClarityPlayer(myPlaylist, {});

        clarity.on('playlistAdvance', function(){
        //do something in here when the playlist advances
        });

You can view a list of all events generated by Clarity here Events

Methods

You can also call functions on Clarity to manipulate it's behavior

    
        var clarity = $('#myPlayer').ttwClarityPlayer(myPlaylist, {});

        //advance to the next song
        clarity.next();

        //change the layout
        clarity.setLayout({
            layout:'album-covers',
            styleType:'blur'
        });

You can view a list of all events generated by Clarity here Methods
